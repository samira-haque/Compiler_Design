## 📖 About

**Compiler Design** is a structured learning repository designed to help students and aspiring software engineers understand how programming languages are translated into machine-executable code. The repository covers core compiler construction concepts, including lexical analysis, syntax analysis, parsing techniques, grammars, semantic analysis, intermediate code generation, optimization, and code generation. Through theoretical explanations, examples, and implementations, learners can develop a strong foundation in language processing and compiler development.

---

- Finite Automata and Regular Expressions
